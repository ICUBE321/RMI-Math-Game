Application: My application is a Math game in which the player must answer randomly generated mathematical equations.

Instructions on how to run application:
To run the game, 

1. compile the .java files using "javac <filename>.java", 

2. on one terminal opened in the Server directory, run "rmiregistry"

2. on another terminal in the Server directory, run "java -Djava.security.policy=policy.txt GameServer" and

3. on a seperate terminal in the Client directory, run "java GameClient localhost".

Follow the game prompts and enjoy...
